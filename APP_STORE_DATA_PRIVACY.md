# App Store データプライバシー設定ガイド

## 📱 App Store Connect 設定手順

このドキュメントは、App Store Connectでのデータプライバシー設定を正確に行うためのガイドです。

---

## 1. 収集されるデータタイプ

App Store Connectで以下の項目を選択してください：

### 連絡先情報
- [x] **メールアドレス** - アカウント作成とログインに使用

### 識別子
- [x] **ユーザーID** - Supabaseで生成される一意のID

### 使用状況データ
- [x] **製品の操作** - レッスンの完了、学習時間
- [x] **アプリ内でのその他の操作** - AI会話、発音練習

### 診断
- [ ] **クラッシュデータ** - 現在収集していない
- [ ] **パフォーマンスデータ** - 現在収集していない

### その他のデータ
- [x] **音声データ** - 発音練習の一時録音（24時間以内に削除）
- [x] **学習進捗データ** - スコア、レベル、実績

---

## 2. データの利用目的

各データタイプごとに以下の目的を選択：

### メールアドレス
- [x] **アプリの機能** - ログインと認証
- [x] **カスタマーサポート** - お問い合わせ対応
- [ ] ~~マーケティング~~ - 使用しない

### ユーザーID
- [x] **アプリの機能** - ユーザーデータの管理
- [ ] ~~分析~~ - 使用しない
- [ ] ~~広告または販促~~ - 使用しない

### 製品の操作データ
- [x] **アプリの機能** - 学習進捗の追跡
- [x] **製品の改善** - 機能改善のため
- [ ] ~~分析~~ - 使用しない

### 音声データ
- [x] **アプリの機能** - 発音評価とフィードバック
- [ ] ~~その他の目的~~ - 使用しない

---

## 3. ユーザーへのリンク

### ユーザーにリンクされるデータ
以下のデータはユーザーアカウントにリンクされます：
- メールアドレス
- ユーザーID
- 学習進捗データ
- 発音評価結果

### ユーザーにリンクされないデータ
- 一時的な音声録音（処理後削除）
- セッションデータ

---

## 4. トラッキング

- [ ] **ユーザーをトラッキングしない** - このアプリはトラッキングを行いません

---

## 5. サードパーティSDK

使用しているサードパーティサービス：

| サービス | データタイプ | 目的 |
|---------|------------|------|
| Supabase | アカウント情報、学習データ | データベース、認証 |
| OpenAI | 会話内容、音声 | AI機能 |
| Azure Speech | 音声録音 | 発音評価 |
| Firebase | 基本的な使用状況 | アプリ配信（Core機能のみ） |

## 6. データセキュリティ

以下のセキュリティ対策を実施：
- HTTPS暗号化通信
- Supabase Row Level Security（RLS）
- 音声データの24時間自動削除
- APIキーの環境変数管理

## 9. App Store Connect入力時のポイント

### ステップ1: データタイプの選択
1. 「App のプライバシー」セクションに移動
2. 「データタイプ」を選択
3. 上記のチェック項目を選択

### ステップ2: データの使用方法
1. 各データタイプごとに「編集」をクリック
2. 該当する利用目的を選択
3. 「ユーザーにリンクされる」を適切に設定

### ステップ3: トラッキング
1. 「いいえ、ユーザーをトラッキングしません」を選択

### ステップ4: プライバシーポリシー
1. プライバシーポリシーURLを入力
2. 言語を「日本語」に設定

## 10. よくある質問と回答

### Q: 音声データは保存されますか？
A: 発音評価のために一時的に処理されますが、24時間以内に自動削除されます。

### Q: 広告は表示されますか？
A: いいえ、このアプリには広告が含まれていません。

### Q: 子供も使用できますか？
A: 13歳以上のユーザーを対象としています。

### Q: データは第三者と共有されますか？
A: 処理のためにOpenAIとAzureに送信されますが、販売や広告目的での共有はありません。

## 11. 重要な注意事項

⚠️ **正確性が重要**: データプライバシーの虚偽申告は審査却下の原因となります

⚠️ **一貫性の維持**: プライバシーポリシーと一致させてください
