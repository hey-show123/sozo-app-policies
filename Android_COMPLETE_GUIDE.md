## 🔧 トラブルシューティング

### よくある審査問題と対策

#### 1. データセーフティの不一致
**問題**: プライバシーポリシーとデータセーフティセクションが一致しない
**解決策**:
```
このガイドの設定を正確に入力し、
プライバシーポリシーの内容と完全一致させる
```

#### 2. 権限の説明不足
**問題**: マイク権限の用途が不明確
**解決策**:
ストア説明文に追加:
```
マイク使用：発音練習機能でのみ使用。
録音は評価後24時間で自動削除されます。
```

#### 3. ターゲットSDKバージョン
**問題**: targetSdkVersionが古い
**解決策**:
`android/app/build.gradle`:
```gradle
android {
    compileSdkVersion 34
    defaultConfig {
        minSdkVersion 21
        targetSdkVersion 34
    }
}
```

#### 4. 64ビット対応
**問題**: 64ビットバージョンがない
**解決策**:
```gradle
android {
    defaultConfig {
        ndk {
            abiFilters 'armeabi-v7a', 'arm64-v8a', 'x86_64'
        }
    }
}
```
